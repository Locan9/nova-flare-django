# ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/Locan9/nova-flare-django.git

# Enter project
cd nova-flare-django

# Create virtual environment
python -m venv venv

# Activate venv
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run server
python manage.py runserver
```
